## 요약

JHipster는 최신 웹 애플리케이션과 마이크로서비스 아키텍처를 빠르게 생성, 개발, 배포할 수 있도록 지원하는 대표적인 개발 플랫폼입니다.

특히 다음 조합에서 강력합니다.

* Spring Boot + React
* Spring Boot + Angular
* Spring Boot + Vue
* Monolith
* Microservices
* OAuth2 / JWT
* Docker / Kubernetes
* Windows / Linux 개발환경

교수님처럼 **Spring Boot 기반 SI/공공 프로젝트, MSA 전환, 관리자 시스템 자동 생성**에 매우 잘 맞습니다.
